## 三、跨语言对比

| 维度 | ArkTS | Java | Swift |
|------|-------|------|-------|
| **多文件模块** | ✅ 同moduleHeader多文件 | ✅ 同包多文件天然支持 | ✅ 同模块多文件天然支持 |
| **export修饰符一致性** | ✅ spec要求 | ✅ Java类访问修饰符一致 | ✅ Swift访问修饰符一致 |
| **顶层语句在多文件中** | ✅ | ❌ (Java所有代码在类内) | ✅ (Swift文件级语句) |

**结论：** ArkTS 的多文件模块概念与 Java 的包、Swift 的模块类似，但 ArkTS 要求 module header 一致性约束。
