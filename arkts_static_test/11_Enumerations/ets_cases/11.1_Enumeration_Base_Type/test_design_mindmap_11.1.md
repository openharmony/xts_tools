# 11.1 Enumeration Base Type - 测试设计思维导图

## 概述
每个枚举都有一个基类类型，可以显式指定或从初始化器推断。

## 推断规则
1. 至少一个成员无初始化器 → 推断为 int（所有初始化器必须可赋值给 int）
2. 全部初始化器可赋值给 int → int
3. 全部初始化器可赋值给 string → string
4. 否则 → 编译错误

## 测试点覆盖

| 规则 | 正向 | 反向 |
|------|------|------|
| 全部无初始化器 → int | 001 | - |
| 全部 int 初始化器 → int | 002 | - |
| 部分 int + 部分无初始化 → int | 010 | - |
| 全部 string → string | 004, 007 | - |
| int + string 混合 | - | 005 |
| 无初始化成员 + string 初始化器 | - | 011 |
| string 推断后缺初始化器 | - | 006 |
| 显式基类（byte~double） | → 11.2 | - |
| 非数值/string 基类 | - | → 11.2 |
