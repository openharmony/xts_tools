## 一、行为差异与规范一致性概览

### 差异 A-4.7.5：剩余参数必须位于参数列表最后（语言设计差异）

**Observation:** ArkTS 要求剩余参数必须位于参数列表最后，而 TypeScript 有相同限制。

**对比：**
| 语言 | 剩余参数位置 |
|------|-------------|
| ArkTS | 必须位于最后 |
| Java | 必须位于最后（varargs） |
| Swift | 必须位于最后 |
| TypeScript | 必须位于最后 |

**影响：** 无负面影响，符合所有主流语言设计。

**评价：** 设计合理，与主流语言一致。
