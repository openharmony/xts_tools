## 一、行为差异与规范一致性概览

### 差异 A-4.7.7：返回类型推断规则（语言设计差异）

**Observation:** ArkTS 的返回类型推断规则与 TypeScript 类似，但更严格。

**对比：**
| 语言 | 返回类型推断 |
|------|-------------|
| ArkTS | 严格（需显式返回类型或明确返回语句） |
| Java | 自动推断（Lambda） |
| Swift | 完整推断（类型推断强大） |
| TypeScript | 基于返回语句推断 |

**影响：** 需要更多类型注解，但提升了类型安全性。

**评价：** 符合 ArkTS 静态化、安全设计哲学，非缺陷。
