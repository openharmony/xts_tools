## 一、行为差异与规范一致性概览

### 差异 A-4.7.3：只读参数语法（语言设计差异）

**Observation:** ArkTS 支持只读参数（参数不可修改），而 Java 不支持此特性。

**对比：**
| 语言 | 只读参数支持 |
|------|-------------|
| ArkTS | ✅ 支持 |
| Java | ❌ 不支持（使用 `final` 局部变量） |
| Swift | ✅ 支持（`let` 参数） |
| TypeScript | ❌ 不支持 |

**影响：** 提升代码安全性，防止意外修改参数。

**评价：** 设计良好，与 Swift 类似。
