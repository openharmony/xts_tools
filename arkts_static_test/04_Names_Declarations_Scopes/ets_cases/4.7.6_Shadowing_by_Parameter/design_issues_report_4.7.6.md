## 一、行为差异与规范一致性概览

### 差异 A-4.7.6：参数遮蔽规则（语言设计差异）

**Observation:** ArkTS 允许参数遮蔽外层作用域的变量，而 Java 允许但警告。

**对比：**
| 语言 | 参数遮蔽 |
|------|---------|
| ArkTS | ✅ 允许（无警告） |
| Java | ✅ 允许（有警告） |
| Swift | ✅ 允许（无警告） |
| TypeScript | ✅ 允许（无警告） |

**影响：** 无重大影响，但可能导致代码可读性下降。

**评价：** 符合主流语言设计，非缺陷。
