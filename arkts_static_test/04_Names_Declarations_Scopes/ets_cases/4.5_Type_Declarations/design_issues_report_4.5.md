### 说明 D-4.5：递归类型限制 — 符合 ArkTS spec 的设计限制

**根据 ArkTS 规范 §4.5 Type Declarations 及 §4.5.1 Type Alias Declaration 中的明确规则：**
- 允许递归的场景：作为数组元素类型 `type A = A[]`、作为泛型类型参数 `type A = C<A>`、`type D = string | Array<D>`
- 禁止的场景：直接自引用 `type E = E`、其余非泛型循环引用 `type F = string | F`、泛型自引用 `type A<T> = A<T>`、无类型参数的泛型别名 `type A<T> = Array<A>`

**所有用例与规范描述一致，该策略属于 ArkTS 有意设计选择，不属于 Spec/实现不一致或 Spec 文档不完整。如需扩展递归能力（如链表递归），应作为语言增强建议提出。</summary>
