# 2.10 Comments - 测试设计思维导图

## 概述
Comment 是补充源代码的文本流，对语法分析不重要。

**Spec 文法：**
```
LineComment: '//' (any character except line separator)* LineSeparator
MultilineComment: '/*' (any character except '*/') '*/'
```

---

### 维度 1：单行注释

#### A. 基本单行注释
- **A1** 基本单行注释（001）
- **A2** 空单行注释（002）
