# 2.10 Comments - ArkTS/Java/Swift 对比报告

## 概述
本报告对比 ArkTS、Java、Swift 三种语言在 comments 测试中的行为差异。

---

## 语言差异对比

| 特性 | ArkTS | Java | Swift |
|------|-------|------|-------|
| 单行注释 | `//` | `//` | `//` |
| 多行注释 | `/* ... */` | `/* ... */` | `/* ... */` |
| 嵌套注释 | ❌ 不支持 | ❌ 不支持 | ❌ 不支持 |
| 文档注释 | `/** ... */` | `/** ... */` | `/// ...` |

---

## 测试用例对比

### 001: 基本单行注释
| 语言 | 代码 | 结果 |
|------|------|------|
| ArkTS | `// comment` | ✅ |
| Java | `// comment` | ✅ |
| Swift | `// comment` | ✅ |

### 004: 基本多行注释
| 语言 | 代码 | 结果 |
|------|------|------|
| ArkTS | `/* comment */` | ✅ |
| Java | `/* comment */` | ✅ |
| Swift | `/* comment */` | ✅ |

### 010: 注释不能嵌套
| 语言 | 代码 | 结果 |
|------|------|------|
| ArkTS | `/* /* nested */ */` | ✅ 编译失败 |
| Java | `/* /* nested */ */` | ✅ 编译失败 |
| Swift | `/* /* nested */ */` | ✅ 编译失败 |

### 011: 单行注释不影响代码执行
| 语言 | 代码 | 结果 |
|------|------|------|
| ArkTS | `let x: int = 1 // comment` | ✅ |
| Java | `int x = 1; // comment` | ✅ |
| Swift | `let x = 1 // comment` | ✅ |

### 012: 多行注释不影响代码执行
| 语言 | 代码 | 结果 |
|------|------|------|
| ArkTS | `/* comment */ let x: int = 1` | ✅ |
| Java | `/* comment */ int x = 1;` | ✅ |
| Swift | `/* comment */ let x = 1` | ✅ |

---

## 关键结论

1. **语法一致性**：注释语法在 ArkTS、Java、Swift 三种语言中完全一致
2. **行为一致性**：注释不影响代码执行的行为在三种语言中一致
3. **嵌套限制**：三种语言都不支持嵌套注释
4. **跨语言差异**：注释是三种语言中差异最小的特性之一
