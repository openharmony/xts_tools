## 三、语言设计差异详情

### 差异 A：文档注释语法

**现象：**
- ArkTS: `/** ... */`
- Java: `/** ... */`
- Swift: `/// ...`

**影响：**
- 文档注释语法层面差异
- 功能等价

**规范依据：**
- ArkTS spec §2.10: 支持 `/** ... */` 文档注释
- Java JLS §3.7: 支持 `/** ... */` Javadoc
- Swift Language Reference: 支持 `/// ...` 文档注释

**分类：** 符合 ArkTS spec 的语言设计差异，非缺陷
