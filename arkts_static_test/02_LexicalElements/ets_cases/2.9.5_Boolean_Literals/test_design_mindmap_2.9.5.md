# 2.9.5 Boolean Literals - 测试设计思维导图

## 概述
Boolean literals 由两个关键字 `true` 和 `false` 表示，类型为 `boolean`。

**Spec 文法：**
```
BooleanLiteral: 'true' | 'false';
```

---

### 维度 1：基本布尔字面量

#### A. 基本值
- **A1** true 字面量（001）
- **A2** false 字面量（001）

### 维度 2：类型推断

#### B. 类型推断
- **B1** true 推断为 boolean（002）
- **B2** false 推断为 boolean（002）
