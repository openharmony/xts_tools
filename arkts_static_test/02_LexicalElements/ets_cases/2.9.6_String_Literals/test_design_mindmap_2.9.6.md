# 2.9.6 String Literals - 测试设计思维导图

## 概述
String literals 由单引号或双引号包围的零个或多个字符组成。支持多种转义序列。

**Spec 文法：**
```
StringLiteral:
  '"' DoubleQuoteCharacter* '"'
  | '\'' SingleQuoteCharacter* '\''
;
EscapeSequence:
  ['"bfnrtv0\\]
  | 'x' HexDigit HexDigit
  | 'u' HexDigit HexDigit HexDigit HexDigit
  | 'u' '{' HexDigit+ '}'
  | ~[1-9xu\r\n]
;
```

## 全覆盖测试矩阵

### 维度 1：基本字符串

#### A. 双引号字符串
- **A1** 普通字符串：`"Hello, world!"`（001）
- **A2** 空字符串：`""`（001）

#### B. 单引号字符串
- **B1** 普通字符串：`'Hello, world!'`（002）
- **B2** 空字符串：`''`（002）

### 维度 2：转义序列

#### C. 基本转义序列
- **C1** 双引号转义：`"He said \"Hi\""`（003）
- **C2** 单引号转义：`'don\'t worry'`（003）
- **C3** 反斜杠转义：`"\\"`（003）

#### D. 控制字符转义
- **D1** 退格符：`"\b"`（004）
- **D2** 换页符：`"\f"`（004）
- **D3** 换行符：`"\n"`（004）
- **D4** 回车符：`"\r"`（004）
- **D5** 水平制表符：`"\t"`（004）
- **D6** 垂直制表符：`"\v"`（004）

#### E. 十六进制转义
- **E1** 十六进制转义：`"\x41"` = "A"（005）

#### F. Unicode 转义
- **F1** 固定 Unicode：`"\u0041"` = "A"（006）
- **F2** 有界 Unicode：`"\u{41}"` = "A"（006）

### 维度 3：类型

#### G. 类型推断
- **G1** 字面量类型推断（007）
- **G2** 运算后变为 string 类型（007）

### 维度 4：compile-fail

#### H. 非法字符串
- **H1** 字符串中包含未转义换行（008）
- **H2** 无效转义序列（009）
- **H3** 未终止的字符串（010）
- **H4** 无效十六进制转义（011）
- **H5** 无效 Unicode 转义（012）

### 维度 5：runtime

#### I. 运行时验证
- **I1** 字符串拼接（013）
- **I2** 转义序列值验证（014）
- **I3** 字符串长度（015）
- **I4** 字符串比较（016）
- **I5** 单引号双引号等价（017）
