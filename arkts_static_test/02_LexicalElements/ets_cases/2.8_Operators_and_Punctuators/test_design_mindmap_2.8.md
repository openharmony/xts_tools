> 📝 重要说明：本测试设计参考 2.1_Use_of_Unicode_Characters 的测试因子checklist设计，补充了作用域组合、类型系统组合、char 特殊场景和 Unicode 在运算符中的应用等覆盖场景。

## 概述
- **Operators**：对值执行操作的 Token，含算术、比较、逻辑、位运算、赋值等。`instanceof` 和 `typeof` 关键字也作运算符。
- **Punctuators**：分隔、完成或组织程序元素的 Token，含逗号、分号、括号等。

| 展开 | `...` |

### 标点（Punctuators）
| Token | 用途 |
|-------|------|
| `(` `)` | 分组、函数调用、参数列表 |
| `[` `]` | 数组、索引 |

| `;` | 语句终止 |

## 测试因子checklist 参考（来自 2.1）

### ✅ 必须正交的因子

#### 1. 支配作用域
- **顶层变量**: unicode 变量的运算符操作
- **函数内局部变量**: unicode 局部变量的运算符操作
- **class method内**: unicode 类方法内变量的运算符操作
- **namespace**: unicode 命名空间成员的运算符操作

#### 2. 类型组合
- **primitive类型字面量**: `c'A'`, `"字符"`, `42`
- **泛型容器**: 运算符在 Map/Set/KT 等容器中的应用
- **数组元素索引**: `arr[运算符]` 的正确性
- **返回值上下文**: 运算符表达式作为函数返回值

#### 3. Unicode 场景（参考 2.1）
- **BMP 字符运算**: \u0041、中文、emoji 等基本多文种平面字符
- **特殊字符运算**: $开头、_开头、ZWJ(U+200D)、ZWNJ(U+200C)
- **Unicode 转义序列**: \uHHHH、\u{} 等转义序列参与运算

#### 4. 类/接口继承关系
- **单层继承**: unicode 子类在运算符继承中的正确性
- **多层继承**: unicode 多级继承链
- **接口实现**: unicode 类实现 unicode 接口中的运算符
