# 2.11 Semicolons - 测试设计思维导图

## 概述
声明和语句通常由行分隔符终止。分号在某些情况下必须使用，用于分隔单行中编写的语法产生式，或避免歧义。

**Spec 文法：**
```
Semicolon: ';'
```

- **A2** 表达式语句由行分隔符终止（002）
