## 结论

### 验证通过的 ArkTS 特性

1. ✅ **bigint 字面量**：`123n` 语法简洁
2. ✅ **bigint 算术运算**：`+`, `-`, `*`, `/`, `%` 支持
3. ✅ **bigint 关系运算**：`<`, `>`, `==` 支持
4. ✅ **bigint 任意精度**：超过 long 最大值
5. ✅ **bigint 与数值类型**：无隐式转换

### ArkTS 与 Java 的差异

- **字面量语法**：ArkTS 有 `n` 后缀，Java 无
- **算术运算符**：ArkTS 用运算符，Java 用方法
- **关系运算符**：ArkTS 用运算符，Java 用 compareTo

### ArkTS 与 Swift 的差异

- **任意精度**：ArkTS 支持，Swift 仅 Int64
- **类型名称**：`bigint` vs `Int64`
- **溢出行为**：ArkTS 无溢出，Swift 有溢出
