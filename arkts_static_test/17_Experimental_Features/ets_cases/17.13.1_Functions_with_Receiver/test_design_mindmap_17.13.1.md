# 17.13.1 Functions with Receiver - 测试设计思维导图

## 概述
接收者函数（Functions with Receiver）是顶层函数，其第一个强制参数名为 `this: Type`。可通过方法调用语法 `receiver.foo(args)` 或普通函数调用语法 `foo(receiver, args)` 调用。`this` 关键字在函数体内引用第一个参数（只读）。只能访问公有成员。支持泛型接收者函数。实例方法优先级高于同名接收者函数。静态分发（无动态分发）。

## 子类型覆盖

### 1. 基本接收者函数定义和调用
- **正向编译**: 定义接收者函数，通过方法调用语法调用
- **正向编译**: 通过普通函数调用语法调用
- **反向编译**: this 不是第一个参数
- **反向编译**: this 参数名称错误
- **运行时**: 验证两种调用语法都正确工作

### 2. 泛型接收者函数
- **正向编译**: 泛型接收者函数定义和调用
- **运行时**: 验证泛型接收者函数运行时正确执行

### 3. 成员访问
- **正向编译**: 通过 this 访问公有成员
- **反向编译**: 通过 this 访问私有成员
- **运行时**: 验证 this 绑定正确

### 4. 命名空间中的接收者函数
- **正向编译**: 命名空间中定义接收者函数
- **运行时**: 验证跨命名空间调用

### 5. 多类型接收者函数
- **正向编译**: 为不同类型定义不同的接收者函数
- **运行时**: 验证各自独立工作
