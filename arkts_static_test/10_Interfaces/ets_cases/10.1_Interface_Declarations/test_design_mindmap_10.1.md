# 10.1 Interface Declarations - 测试设计思维导图

## 概述
Interface declaration specifies a new named reference type with properties and methods, no instance variables. Cannot be instantiated directly.

## 子类型覆盖
### 1. Basic interface declaration
- 正向编译: 简单接口声明
- 运行时: 实现接口的类可正常使用

### 2. Generic interface
- 正向编译: 带类型参数的接口

### 3. Interface with extends
- 正向编译: 继承父接口
