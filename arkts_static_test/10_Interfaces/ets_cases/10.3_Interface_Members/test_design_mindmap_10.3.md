# 10.3 Interface Members - 测试设计思维导图

## 概述
Interface members are properties or methods. Member names must be unique. Method names must not clash with Object's public methods.

## 子类型覆盖
### 1. Property members
- 正向编译: 字段形式属性

### 2. Method members
- 正向编译: 抽象方法声明

### 3. Name uniqueness
- 反向编译: 同名属性和方法

### 4. Object method clash
- 反向编译: 方法名与 Object 的 public 方法冲突
